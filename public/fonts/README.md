# Polices à télécharger

Télécharger les fichiers .woff2 suivants et les placer dans ce dossier :

1. **Plus Jakarta Sans Variable** : https://fonts.google.com/specimen/Plus+Jakarta+Sans
   → Fichier : `PlusJakartaSans-Variable.woff2`

2. **Inter Variable** : https://fonts.google.com/specimen/Inter
   → Fichier : `Inter-Variable.woff2`

Les deux polices sont open source (OFL license).
